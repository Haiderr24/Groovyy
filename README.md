# Groovy 🎵

**Discover music that moves you.**

A TikTok-style music discovery app with infinite scroll, personalized feeds, and beautiful audio visualization.

---

## Features

- **Infinite scroll** through curated music from your favorite genres
- **Audio previews** with real-time visualization
- **Save favorites** to your personal collection
- **6 beautiful themes** to match your vibe
- **Smart curation** - mix of trending hits and classic favorites

## Tech Stack

Next.js 15 • React 19 • MongoDB • iTunes API • Tailwind CSS

## Quick Start

```bash
npm install
npm run dev
```

See [SETUP.md](./SETUP.md) for detailed instructions.

## Planned Enhancements

### Personal Analytics Dashboard
A comprehensive music analytics system to help users understand their listening habits:

- **Listening Statistics** - Top artists, tracks, and albums with play counts
- **Genre Distribution** - Visual breakdowns of music taste (pie/bar charts)
- **Listening Timeline** - Activity patterns and listening history over time
- **Discovery Metrics** - Track exploration of new artists and genres
- **Listening Patterns** - Heatmaps showing listening habits by time of day/week
- **Music Taste Profile** - Personalized insights and "year in review" summaries
- **Data Visualizations** - Interactive charts using Chart.js or Recharts

### AI-Powered Playlist Generator
Smart playlist creation using natural language and AI:

- **Natural Language Input** - Create playlists with prompts like "upbeat 2000s pop for working out"
- **Intent Recognition** - AI parsing to understand mood, genre, and activity preferences
- **Smart Matching** - Automatic genre and attribute matching from iTunes catalog
- **Themed Playlists** - Activity-based, mood-based, and custom-themed collections
- **OpenAI Integration** - Leverage GPT models for intelligent music curation

**Tech Stack Additions:**
- Chart.js/Recharts for data visualization
- OpenAI API for natural language processing
- MongoDB aggregation pipelines for analytics
- Additional API routes for statistics and playlist generation
